# 🌐 Module 2 - 웹사이트 과제

이 저장소는 내가 만든 HTML/CSS 연습 과제를 담고 있어요.  
간단한 웹사이트 레이아웃을 만들면서 HTML 구조와 CSS 스타일링을 연습했어요.

## 📁 파일 설명

- `index.html` – 기본 웹페이지 구조 (제목, 본문, 이미지 등)
- `style.css` – 웹사이트에 적용되는 디자인 (색상, 폰트, 배치 등)

## 💡 배운 점
- HTML 태그 사용법 (예: `<div>`, `<header>`, `<img>` 등)
- CSS 기본 스타일 작성
- 웹페이지를 깔끔하게 구성하는 법
